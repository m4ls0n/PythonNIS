## Инструкция по применению миграций

Чтобы инициализировать бд и применить структуру (а также загрузить моки), надо выполнить эти шаги из корня проекта:

1. **Поднять контейнер с бд:**
   ```bash
   make db
   # или напрямую через docker-compose: docker-compose up -d
   ```

2. **Создать файлы миграций (если были изменения в моделях):**
   ```bash
   make migrations
   # или напрямую: python3 backend/manage.py makemigrations
   ```

3. **Применить миграции к бд:**
   ```bash
   make migrate
   # или напрямую: python3 backend/manage.py migrate
   ```

После выполнения `make migrate` в базе будут созданы все нужные таблицы и загружены начальные моки для тестирования
