# HW 7

## Запуск

1. **Запуск контейнеров:**
   ```bash
   make up
   ```

2. **Применение миграций:**
   ```bash
   make migrate
   ```

3. **Запуск автотестов:**
   ```bash
   make test
   ```

api по адресу: http://127.0.0.1:8000/  
swagger: http://127.0.0.1:8000/swagger/


## Что реализовано

### Авторизация и права 
- подключены TokenAuthentication и SessionAuthentication
- настроены базовые права: IsAuthenticatedOrReadOnly (читать могут все, изменять только авторизованные)
- написан кастомный класс IsAuthorOrReadOnly (редактировать и удалять посты, комментарии может только их создатель)
- внедрено автосохранение текущего пользователя как автора при создании постов и комментариев (perform_create)

### Тесты 
- написаны тесты с помощью APITestCase
- покрыт базовый crud
- проверена работа прав доступа (анонимы, авторы, чужие пользователи)
- написаны тесты для кастомных action методов (lightweight и stats)

### Сборка под Docker Compose 
- проект полностью изолирован в контейнерах
- настроен docker-composeyml для связи PostgreSQL и Django
- написан удобный Makefile для проксирования команд внутрь контейнеров (make up, make test, make migrate)
