# HW5 Django Project 

### 1. Подготовка

```bash
make venv
make install
```

### 2. Запуск бд

```bash
make db
```

### 3. Миграции и админ

```bash
make migrate
make superuser
```

### 4. Запуск сервера
```bash
make run
```
адрес: http://127.0.0.1:8000
