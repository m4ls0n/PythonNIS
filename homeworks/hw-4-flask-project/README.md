# HW 4: Flask Project

Учебный проект на Flask. ДЗ по НИСу.

## Старт

Для управления проектом используется `Makefile`

1. **Создание виртуального окружения:**
   ```bash
   make venv
   ```

2. **Установка зависимостей:**
   ```bash
   make install_requirements
   ```

3. **Запуск сервера:**
   ```bash
   make run
   ```
   Сервер здесь: http://127.0.0.1:5000

4. **Запуск тестов:**
   ```bash
   make test
   ```

## Выполненные требования

Работа выполнена на **10 баллов**. Реализовано:

- **1 балл**: Созданы README.md, requirements.txt
- **2 балла**: Инициализировано Flask-приложение в server.py. В requirements.txt добавлена зависимость для flask
- **3 балла**: Добавлен endpoint /author, возвращающий JSON
- **4 балла**: Настроен конфиг через .env (чтение порта с помощью python-dotenv)
- **5 баллов**: Создан Makefile для команд venv, install, run.
- **6 баллов**: Добавлена ручка, которая умеет складывать числа из параметров
- **7 баллов**: Бизнес-логика вынесена из server.py в отдельный модуль controllers.py
- **8 баллов**: Контроллер покрыт unit-тестами с использованием pytest
- **9-10 баллов**: Весь код покрыт тайпингами
